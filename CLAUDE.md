# BlenderPlugins

## Source of truth

All edits go to files in this repo (`c:\Users\WXP\Documents\GitHub\BlenderPlugins\`).

Never edit files in Blender's addons folder (`C:\Users\WXP\AppData\Roaming\Blender Foundation\...`) directly. Those are deployment copies. After changing a file here, the user handles copying/installing it into Blender.
