## Bone Renamer v1.2

**Purpose**: Rename armature bones between different naming conventions and optionally translate Japanese bone names to English.

**Key features**
- **Multi-format renaming**: Converts bone names between multiple formats (MMD English/Japanese, XNALara, DAZ/Poser, Rigify, Sims 2, Motion Builder, 3ds Max, BEPu, and custom/unknown).
- **Finger support**: Optional inclusion of finger bones in the renaming pass.
- **Japanese → English translation**: Replaces common Japanese bone name fragments with English equivalents.
- **Viewport helper**: Toggle visibility of bone names for all armatures in the scene.

**UI location**
- `3D Viewport` → Sidebar (`N`) → `Animation` tab → `Bone Renamer` panel.

**Basic usage**
1. Select an armature and pick it with the eyedropper in **Armature**.
2. Choose **From** and **To** formats (e.g. *MMD English* → *Rigify*).
3. Optionally enable **Include Fingers**.
4. Click **Rename Bones**.
5. (Optional) Use **Translate Japanese Names** on MMD-style rigs.
6. Use **Toggle Names Display** to quickly show/hide bone names in the viewport.
